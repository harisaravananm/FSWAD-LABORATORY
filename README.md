## Full Stack Web Application Development Laboratory

A comprehensive hands-on lab where students build complete web applications using VS Code with productivity-enhancing extensions. The course progresses from basic JavaScript to full MEAN stack development.

### **Required Software & Tools**

1. **VS Code** (Primary IDE)
   - Download from code.visualstudio.com
   - Lightweight, fast, and cross-platform

2. **Essential Extensions for High Accuracy:**
   - **Prettier** - Code formatter for consistent styling
   - **ESLint** - JavaScript linting for error detection
   - **Live Server** - Local development server with live reload
   - **GitLens** - Enhanced Git capabilities
   - **Thunder Client** - API testing (alternative to Postman)
   - **Angular Language Service** - AngularJS/Angular support
   - **MongoDB for VS Code** - Database management
   - **Node.js Extension Pack** - Node/Express development tools

3. **Runtime & Frameworks:**
   - **Node.js** (v16 or higher) - JavaScript runtime
   - **npm** - Package manager (comes with Node.js)
   - **AngularJS/Angular** - Frontend framework
   - **Express.js** - Backend framework
   - **MongoDB** - NoSQL database

4. **Version Control:**
   - **Git** - Version control system
   - **GitHub account** - For repository hosting

### **Lab Structure (10 Experiments)**

**Phase 1: JavaScript Fundamentals (Experiments 1a-1c)**
- Basic DOM manipulation
- Event handling
- Dynamic content updates

**Phase 2: AngularJS Frontend (Experiments 2a-5)**
- Component-based architecture
- Data binding and forms
- Building interactive applications (calculator, to-do list, weather app, stocks)

**Phase 3: Backend with Express.js (Experiments 6-7)**
- RESTful API development
- Form handling and authentication
- Server-side routing

**Phase 4: Database Integration (Experiments 8-9)**
- MongoDB operations with Node.js
- CRUD operations using Angular + MongoDB

**Phase 5: Full Stack MEAN Application (Experiment 10)**
- MongoDB + Express + Angular + Node.js
- Complete task manager application

### **How to Use This Lab**

1. **Setup Phase:**
   - Install VS Code and all required extensions
   - Install Node.js and verify with `node --version`
   - Install MongoDB locally or use MongoDB Atlas (cloud)
   - Clone/create GitHub repository for version control

2. **For Each Experiment:**
   - Create a new folder in your workspace
   - Use Live Server extension for frontend experiments
   - Use integrated terminal for Node.js/Express projects
   - Test APIs using Thunder Client extension
   - Commit changes using GitLens after each milestone

3. **Best Practices:**
   - Enable auto-save in VS Code
   - Use Prettier to format on save
   - Let ESLint catch errors as you code
   - Use Git for version control from day one
   - Test incrementally, don't wait until completion
